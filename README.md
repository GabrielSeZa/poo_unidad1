\## Descripcion del trabajo



El proyecto implementa un sistema audiovisual en \*\*Java\*\*, aplicando los principios de la \*\*Programacion Orientada a Objetos (POO)\*\*: \*\*herencia\*\*. \*\*polimorfismo y \*encapsulamiento\*\*.

El sistema permite gestionar distintos tipos de contenidos, como peliculas, series, documentales, podcasts y transmisiones en vivo.



\##Objetivos 



* Aplicar herencia y polimorfismo en un sistema funcional.
* Crear y manipular objetos de todas las clases.
* Comprobar relaciones entre clases mediante pruebas ejecutadas en consola.
